# Bare-Metal-Drivers

## Objective
Show the use of three drivers: adc, gpio output and uart transmission. 

There are two versions: 
P1 with the drivers in assembly language,
P2 with the drivers in C language.

### Skills Learned

- Bare Metal Drivers

### Tools Used

- STM32CubeIDE

### Languages

- Assembly
- C

### Hardware

- STM32F4
- UART to USB adapter
